# BackSlash-Shell-Sidebar
BackSlash Shell Sidebar for KDE Plasma Desktops


In order to install the BackSlash Shell Sidebar follow the following steps as you copy ans paste the codes in a Terminal window.

## Cloning Repository

First Clone this repository using:

```
git clone https://github.com/luciferreeves/BackSlash-Shell-Sidebar
```

## Renaming Folder

Rename the Cloned folder to "**com.backslash.sidebar**". To do this type in terminal:

```
mv BackSlash-Shell-Sidebar com.backslash.sidebar
```

## Installing

Installing this sidebar requires administrator access, so please get ready with your root password and then type:

```
sudo mv com.backslash.sidebar /usr/share/plasma/plasmoids
```
